# Project01 - Hello World

## คำอธิบายโปรเจกต์
นี่เป็นโปรเจกต์อย่างง่ายที่แสดงเพจ HTML พื้นฐานพร้อมข้อความ "Hello, World v4"

## โครงสร้างของไฟล์
```
Project01/
├── index.html          # ไฟล์ HTML หลัก
└── README.md           # ไฟล์คำอธิบายนี้
```

## วิธีการใช้งาน
1. เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ของคุณ
2. จะแสดงหน้า HTML ที่มีข้อความ "Hello, World v4"

## เทคโนโลยีที่ใช้
- HTML5
- CSS (ยังไม่ได้ใช้งานในเวอร์ชั่นปัจจุบัน)
- JavaScript (ยังไม่ได้ใช้งานในเวอร์ชั่นปัจจุบัน)
